# Assets Directory

This directory contains image and media assets for your about-me page.

## Required Images

To customize this template, you'll need to add the following images:

### Profile Image

- **File:** `profile.jpg` or `profile.png`
- **Size:** 400x400px (square aspect ratio)
- **Purpose:** Main hero section profile photo
- **Tips:** Use a high-quality, professional headshot with good lighting

### Project Images

- **Files:** `project1.jpg`, `project2.jpg`, `project3.jpg`
- **Size:** 600x400px (3:2 aspect ratio)
- **Purpose:** Showcase images for your featured projects
- **Tips:** Use screenshots, mockups, or representative images of your work

### Resume

- **File:** `resume.pdf`
- **Purpose:** Downloadable resume/CV for visitors
- **Tips:** Keep it updated and ensure it's properly formatted

## Image Optimization Tips

1. **File Size:** Keep images under 500KB for web performance
2. **Format:** Use JPEG for photos, PNG for graphics with transparency
3. **Compression:** Use tools like TinyPNG or ImageOptim to reduce file sizes
4. **Alt Text:** Update the alt attributes in HTML for better accessibility

## Placeholder Images

If you don't have images ready, you can use placeholder services:

- [Unsplash](https://unsplash.com/) - Free high-quality photos
- [Placeholder.com](https://placeholder.com/) - Simple placeholder images
- [Lorem Picsum](https://picsum.photos/) - Random placeholder images

## File Structure

```
assets/
├── README.md (this file)
├── profile.jpg         # Your main profile photo
├── project1.jpg        # Project 1 screenshot/image
├── project2.jpg        # Project 2 screenshot/image
├── project3.jpg        # Project 3 screenshot/image
└── resume.pdf          # Your resume/CV
```

## Customization Notes

- Update the `src` attributes in `index.html` to match your actual file names
- Consider adding a favicon by including `favicon.ico` in this directory
- You can add more project images by following the naming convention
