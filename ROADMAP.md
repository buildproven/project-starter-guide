# Roadmap

Strategic direction and future plans for Project Starter Guide.

**Last Updated**: 2025-12-14

## Vision

Provide production-ready starter templates that help developers launch projects faster with best practices built-in.

## Current State (v2.2.0)

- 4 templates: about-me-page, api-service, mobile-app, saas-level-1
- All P0/P1/P2 backlog items complete
- 0 security vulnerabilities across all templates
- Comprehensive testing (90%+ coverage)
- CI/CD with automated quality gates

## Strategic Priorities

### Q1 2025: Monetization Foundation

| Initiative         | Description                    | Status  |
| ------------------ | ------------------------------ | ------- |
| Video tutorials    | Setup guides for each template | Planned |
| Starter Pack       | Bundled templates + tutorials  | Planned |
| Community feedback | NPS surveys, feature requests  | Planned |

### Q2 2025: Platform Expansion

| Initiative              | Description                    | Status |
| ----------------------- | ------------------------------ | ------ |
| Template generator      | Interactive customization tool | Idea   |
| Python/FastAPI template | Expand language coverage       | Idea   |
| Go/Fiber template       | High-performance API option    | Idea   |

### Q3 2025: Enterprise Features

| Initiative             | Description                | Status |
| ---------------------- | -------------------------- | ------ |
| Multi-tenancy template | SaaS Level 2               | Idea   |
| Admin dashboard        | Common SaaS patterns       | Idea   |
| Analytics integration  | Built-in product analytics | Idea   |

## Non-Goals

- Building a full framework (templates should remain simple starters)
- Supporting every tech stack (focus on proven, popular choices)
- Enterprise consulting services (focus on self-service products)
